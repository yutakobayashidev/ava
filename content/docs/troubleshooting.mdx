## よくある質問

### Slackに通知が届かない

**考えられる原因**:

1. **通知先チャンネルが未設定**
   - [設定ページ](/settings/channel)でチャンネルを設定してください

2. **ボットの権限が不足**
   - Slackアプリの設定から、必要な権限（`chat:write`, `chat:write.public`）が付与されているか確認してください

### コーディングエージェントから接続できない

**確認事項**:

1. **MCPサーバーのURLが正しいか**

   Claude Codeの場合（`.mcp.json`）:

   ```json
   {
     "mcpServers": {
       "ava": {
         "type": "http",
         "url": "https://ava-dusky-gamma.vercel.app/mcp"
       }
     }
   }
   ```

   Cursorの場合（`.cursor/mcp.json`）:

   ```json
   {
     "mcpServers": {
       "ava": {
         "type": "http",
         "url": "https://ava-dusky-gamma.vercel.app/mcp"
       }
     }
   }
   ```

2. **認証が完了しているか**
   - 初回接続時にブラウザで認証を完了してください
   - 認証トークンの有効期限が切れている場合は、再認証が必要です

3. **ネットワーク接続**
   - ファイアウォールやプロキシが接続をブロックしていないか確認してください

### タスクが開始されない

**確認ポイント**:

1. **Avaとの連携が完了しているか**
   - コーディングエージェントで「利用可能なMCPツールを教えて」と尋ね、Avaのツールが表示されるか確認

2. **タスク開始の指示が明確か**
   - 「〜を始めます」「〜を修正します」など、明確にタスクの開始を示す表現を使ってください

### 進捗が更新されない

コーディングエージェントで作業を進めていれば、自動的に進捗が記録されます。もし更新されない場合：

1. **コーディングエージェントとの接続を確認**
   - MCPサーバーへの接続が維持されているか確認してください

2. **手動で進捗を報告**
   - コーディングエージェントに「進捗を報告して」と指示することもできます

### 詰まり状態が解除されない

詰まり状態は、新しい進捗やタスク完了時に自動的に解除されます。手動で解除する必要はありません。

## エラーメッセージ

### "Unauthorized" エラー

認証トークンが無効または期限切れです。

**解決方法**:

1. コーディングエージェントを再起動
2. ブラウザで再認証
3. 必要に応じて、Avaからログアウト→再ログイン

### "Channel not found" エラー

通知先チャンネルが見つかりません。

**解決方法**:

1. [設定ページ](/settings/channel)で通知先を再設定
2. チャンネルが削除されていないか、Slackで確認

### "Insufficient permissions" エラー

ボットに必要な権限がありません。

**解決方法**:

1. Slackワークスペースの管理者に連絡
2. ボットを再インストールして、すべての権限を付与

## サポート

上記で解決しない問題がある場合は、以下の情報を添えてサポートにお問い合わせください：

- 発生している問題の詳細
- エラーメッセージ（あれば）
- 実行した手順
- 使用環境（OS、コーディングエージェントの種類とバージョンなど）

## よくあるご要望

### 複数のワークスペースで使いたい

現在、1つのアカウントで1つのSlackワークスペースに対応しています。複数ワークスペースのサポートは今後の予定です。

### 通知先を途中で変更したい

[設定ページ](/settings/channel)から、いつでも通知先チャンネルを変更できます。

### 過去のタスク履歴を見たい

コーディングエージェントで「完了したタスクの一覧を見せて」と尋ねると、過去のタスク履歴を確認できます。
